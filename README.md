Background: Stage IIB conventional osteosarcoma exhibits highly heterogeneous responses to neoadjuvant chemotherapy, necessitating reliable, noninvasive pre-treatment evaluation systems. Methods: This retrospective study of 198 patients utilized pre-treatment T2-weighted magnetic resonance imaging (MRI) and baseline clinical data. Twelve classical convolutional neural network architectures were benchmarked to identify the optimal imaging backbone. This selected radiomics score was then integrated with core clinical variables using logistic regression to construct a multimodal model. Gradient-weighted Class Activation Mapping++ (Grad-CAM++) and SHapley Additive exPlanations (SHAP) algorithms provided image- and feature-level interpretability. Results: DenseNet121 emerged as the optimal imaging model, achieving an area under the curve (AUC) of 0.797 in the training set and 0.701 in the test set. With the integration of key clinical features, the combined model demonstrated significantly enhanced predictive performance, yielding an AUC of 0.855 in the training set and 0.829 in the test set. Grad-CAM++ visual mapping confirmed the model's precise focus on pathologically critical solid medullary infiltration and extraosseous soft tissue invasion zones. Conclusion: Integrating deep learning radiomics with key clinical features provides a noninvasive tool for the individualized prediction of neoadjuvant chemosensitivity in osteosarcoma.
